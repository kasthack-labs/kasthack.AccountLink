Линковка аккаунтов:
	Откройте template.txt в Notepad++ или редакторе, совместимом по regex'ам
	Выполните команду "Заменить" со следующими параметрами:
		'{USERNAME}' -> <ваше имя пользователя> (поиск: обычный)
		'{GOOGLE}' -> <ваше id G+> (поиск: обычный)
		'{GKOD}' -> <ваш id Govnokod> (поиск: обычный)
		'{FLICKR}' -> <символы после flickr.com/photos/ в вашем профиле> (поиск: обычный)
		'{USERSTYLES}' -> <ваш id на userstyles>(поиск: обычный)
		'{EMAIL}' -> <ваш email>(поиск: обычный)
		'{BTC}' ->  <ваш BTC>(поиск: обычный)
		'{LTC}' ->  <ваш LTC>(поиск: обычный)
	Вставьте получившийся текст в поле "Обо мне" на  свою web-страницу		
	Выполните команду "Заменить" со следующими параметрами:
		'<(\/?)h2>' -> '<\1strong>' (поиск: regex)
	Вставьте получившийся текст в поле "Обо мне" на  Goodreads
	Выполните команду "Заменить" со следующими параметрами:
		'<(\/?)strong>' -> '<\1b>' (поиск: regex)
	Вставьте получившийся текст в поле "Обо мне" на  UserStyles, DeviantArt, LJ, D3, HabraHabr
	Выполните команду "Заменить" со следующими параметрами:
		'<a href=".*">.*</a>' -> '[url=\1]\2[/url]' (поиск: regex)
		'<(\/?)b>' -> '[\1b]' (поиск: regex)
	Вставьте получившийся текст в поле "Обо мне" на  Google+, Last.FM, MyAnimeList
	Выполните команду "Заменить" со следующими параметрами:
		'\r\n' -> '[br]\r\n' (поиск: расширенный)
		'([b].*[/b])' - >'[br]\1[/br]' (поиск: regex)
	Вставьте получившийся текст в поле "Обо мне" на linux.org.ru
	Выполните команду "Заменить" со следующими параметрами:
		'[br]' -> '' (поиск: обычный)
		'[url=http://(.*)](.*)[/url]' -> '\2: www.\1' (поиск: regex)
		'[/?b]' -> '' (поиск: regex)
	Вставьте получившийся текст в поле "Обо мне" на FB
	Выполните команду "Заменить" со следующими параметрами:
		': www.' -> ': ' (поиск: обычный)
	Вставьте получившийся текст в поле "Обо мне" на VK
	Вставьте ссылку на свою web-страницу в свои профиль на Github, Ohloh, Compilr
